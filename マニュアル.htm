【GameObject】

IGameObjectは勝手にゲームループで動きます。通常のクラスや型と同じように扱ってください。

NewGOを使う必要はないです。

 

NewGOとDeleteGOが追加されました。new・delete演算子を直接使うより、トラブルが起きにくくなると思うよ。

※DeleteGOでオブジェクトが完全に削除されるタイミングは、そのフレームの全GOのPostUpdateが終わってからです。

※即座に削除するInstantDeleteGOもあります。

 

FindGOとQueryGOsが追加されました。IGameObjectのメンバに名前を設定するSetName関数があります。

 

【カメラ】

GameObj::PerspectiveCamera等のカメラを作り、SetMainCameraで設定してください。

デフォルトではカメラが存在しません。

【3Dモデル】

ボーンが存在するモデルは必ずtksファイルを作成してください。

マテリアル

いろいろ設定できる。

GameObj::CSkinModelRender model;

model.Init(L"Resource/modelData/unityChan.cmo");

 

//モデルのマテリアルを検索する

model.GetSkinModel().FindMaterial(

        //マテリアルが見つかった際に実行する処理

        [&](ModelEffect* mat) {

                if (mat->EqualMaterialName(L"Material #76")) {//マテリアルの名前が"Material #76"か？

                        mat->SetAlbedoScale({ 1.0f,0.05f,0.05f,1.0f });     //色を赤くする

                        mat->SetLightingEnable(false);                    //ライティングを無効化

                        mat->SetEmissive({ 0.0f,0.0f,0.5f });              //自己発光色を設定

                        mat->SetAlbedoTexture(tex);                       //テクスチャ変更

                }

        }

);

 

//CSkinModelRenderごとに設定したい場合

model.GetSkinModel().FindMaterialSetting(

        [&](MaterialSetting* mat) {

                mat->SetAlbedoScale({ 1.0f,0.05f,0.05f,1.0f });

        }

);

 

【アニメーション】

サンプルです

GameObj::CSkinModelRender m_model;

 

enum EnAnimationClip {

        enAnimationClip_run,     //走りアニメーション。

        enAnimationClip_walk,     //歩きアニメーション。

        enAnimationClip_num,     //アニメーションクリップの総数。

};

AnimationClip m_animationClips[enAnimationClip_num];

 

//読み込み

m_animationClips[enAnimationClip_run].Load(L"Resource/animData/run.tka", true);//trueつけるとループ再生でロード

m_animationClips[enAnimationClip_walk].Load(L"Resource/animData/walk.tka", false, enZtoY);//第三引数でアニメーションの上方向軸を変換できます

m_animationClips[enAnimationClip_walk].SetLoopFlag(true);//関数でループ再生に設定

 

m_model.Init(L"Resource/modelData/unityChan.cmo", m_animationClips, enAnimationClip_num, enFbxUpAxisY);

 

//再生

m_model.GetAnimCon().Play(enAnimationClip_run, 0.3f); //m_animationClips[enAnimationClip_run]を再生。0.3秒の補完時間。

m_model.GetAnimCon().SetSpeed(0.5f);//再生速度変更

 

【アニメーションイベント】

アニメーションイベントの付け方です。

Maxでボーンのルートオブジェクトに作成したノートがアニメーションイベントとして出力されます。

 

(1)  ドープシートを開く

 

(2)  ボーンのルートオブジェクト(ボーンの一番上の階層)を選択

 

(3)  ノートトラックを追加

 

(4)  追加されたノートトラック(”ノート”)を選択&”キーの追加/除去”を選択

 

(5)  イベントを作成したいフレームをクリックしてノートを作成

 

(6)  作成したノートを右クリックでイベントの名前を編集してください。

 

(7)  DemolisherWeapon付属のtkExporterでアニメーションを出力してください。

 

【コリジョンシステム】

Sample/CollisionSample にサンプルがあります。

グループとマスクについて

CCollisionObjにグループとマスクを設定することができます。

これは、判定するコリジョン同士の組み合わせを設定するものです。

(例:プレイヤーの攻撃コリジョンは、エネミーの喰らいコリジョンとの当たり判定を実行する。しかし他のコリジョンとは判定しない。)

(ｺﾘｼﾞｮﾝA.group &ｺﾘｼﾞｮﾝB.mask) != 0 && (ｺﾘｼﾞｮﾝB.group &ｺﾘｼﾞｮﾝA.mask) != 0

上記の条件を満たす場合のみ、コリジョン同士の当たり判定が実行されます。

 

デフォルトでグループには1、マスクには0xFFFFFFFFが設定されています。

//SE

SuicideObj::CSE* se = NewGO<SuicideObj::CSE>(L"Resource/sound/LRtest.wav");

se->Play(); //再生(再生が終わると削除されます)

//3D再生

se->SetPos(pos);//音の位置

se->SetDistance(100.0f);//音が聞こえる範囲

se->Play(true); //第一引数をtrue

 

//BGM

SuicideObj::CBGM* bgm = NewGO<SuicideObj::CBGM>(L"Resource/sound/ 01 創世紀.wav");

bgm->Play(false,true); //ループで再生(第二引数)

bgm->Stop(); //再生を止める(その後削除される)

bgm->Pause(); //一時停止。Play()で再生再開(引数は意味ない)

se->SetVolume(1.0f); //1.0fがデフォ

//ファイル自体への音量を設定

//※既に初期化されているCSE・CBGMの音量は変わりません

GetWAVSettingManager().Load(L"Resource/sound/LRtest.wav")->volume = 1.0f;

//マスターボリュームを設定

SetMasterVolume(1.0f);

//SE全体のボリュームを設定

SuicideObj::CSE::GetSubmixVoice()->SetVolume(1.0f);

//BGM全体のボリュームを設定

SuicideObj::CBGM::GetSubmixVoice()->SetVolume(1.0f);
